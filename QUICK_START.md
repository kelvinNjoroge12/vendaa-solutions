# Quick Start Guide

Get up and running with Vendaa Solutions in 3 simple steps!

## Step 1: Install Dependencies

```bash
npm install
```

This will install all required packages including:
- React & React DOM
- TypeScript
- Vite
- Tailwind CSS
- All UI component libraries

## Step 2: Start Development Server

```bash
npm run dev
```

Your app will be available at: `http://localhost:5173`

## Step 3: Start Building!

The project is now ready. You can:

- Edit components in `src/components/`
- Modify sections in `src/sections/`
- Add new types in `src/types/`
- Customize styles in `src/index.css`

## Common Commands

```bash

# Production
npm run build        # Build for production
npm run preview      # Preview production build

# Code Quality

```
